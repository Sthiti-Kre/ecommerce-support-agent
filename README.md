## CI/CD

This repository includes GitHub Actions workflows:
- **CI Pipeline** (`.github/workflows/ci.yml`): Runs linting and tests on every push and pull request.
- **CD Pipeline** (`.github/workflows/cd.yml`): Builds a Docker image and deploys to Google Cloud Run on merge to `main`.

### Setting up CD Pipeline

To enable auto-deployment, add these GitHub repository secrets:
- `GCP_PROJECT_ID` - Your Google Cloud project ID
- `GCP_SA_KEY` - Service account key JSON with Cloud Run deploy permissions
- `GCP_REGION` - Target region (e.g., `us-central1`)
